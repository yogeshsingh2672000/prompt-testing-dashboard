# Code of Conduct

## Our Pledge

We want Promitly to be a welcoming, respectful, and collaborative open-source project for contributors of all backgrounds and experience levels.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Being respectful and constructive in feedback
- Assuming good intent and communicating with clarity
- Sharing context that helps others learn and contribute
- Focusing criticism on ideas, code, and decisions rather than people

Examples of unacceptable behavior:

- Harassment, discrimination, or personal attacks
- Trolling, intimidation, or deliberately hostile language
- Publishing private information without permission
- Repeated bad-faith behavior that makes collaboration unsafe

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, pull requests, and other contributions that violate this code of conduct.

## Scope

This code of conduct applies in GitHub issues, pull requests, discussions, and other community spaces connected to the project.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainers privately. Security-related concerns should follow the guidance in `SECURITY.md`.
